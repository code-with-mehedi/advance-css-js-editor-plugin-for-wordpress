Add Custom CSS and JS script to your website from the backend..

== Description ==
With this plugin you can simply add custom css and js code to customize your WordPress theme's appearance. 


= Features =
* CodeMirror editor with auto suggest code and syntax highlighting
* Three types of editor's themes
* Add unlimited code and file
* Add CSS and JS to the frontend area or admin area


== Installation ==
1. Download and unzip the plugin. Upload the unzipped folder to the wp-contents/plugins folder of your WordPress installation.
2. Active the plugin from the WordPress Plugins administration page.
3. OR, Got to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'Advance Custom Scripts'.
4. Select and click 'Install'.
5. Activate the plugin.
